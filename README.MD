# Maven
### pom.xml
```
<dependency>
    <groupId>me.openautonomousconnection.protocol</groupId>
    <artifactId>Protocol</artifactId>
    <version>1.0-SNAPSHOT</version>
    <scope>compile</scope>
</dependency>
```

### Repository:
```
<repository>
    <id>github</id>
    <url>https://maven.pkg.github.com/open-autonomous-connection/*</url>
    <snapshots>
        <enabled>true</enabled>
    </snapshots>
</repository>
```
